# Node.Js Note app

## Description

A simple console node.js note taking app 

## Features

### Create 
### Read 
### List
### Remove


## User guide

### Start Up 
1. Open in Code Editor 
2. npm install

### Create Note
In the terminal run the following command to add a note

node app.js add --title'_your title here_' --body'_your note body here_'

### Read Note
In the terminal run the following command to read a note

node app.js read --title'_title of note you would like to read_' 

### List Note
In the terminal run the following command to list all notes

node app.js list 

### Remove Note
In the terminal run the following command to remove a note

node app.js remove --title'_title of note you would like to remove_'
